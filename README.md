## Description

Service for crypto trading and operate with crypto data

## Installation

```bash
$ yarn install
```

## Running the app

### Docker (Recommend)

```bash
$ docker-compose up -d
```

### yarn
```bash
# Development
$ yarn start:dev

# Production
$ yarn build
$ yarn start
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
